Sunbit Instructions

Every word is a name of a person from the Sunbit technology org family. Since worle words are always exactly 5 letters long, the logic goes like this:

  • If the name is exactly 5 letters long, you can guess the name directly. This is true for first, middle and last names.
  • If the first name is shorter than 5 letters, you can add letters from the last name to make it 5 letters long.
  • If the name is longer than 5 letters, the first 5 letters are the word for our game.
for example:
  • Doron Sinai -> SINAI, or DORON
  • Marcelo Weissman -> WEISS, or MARCEL
  • Yael Karmon -> YAELK, or KARMON
